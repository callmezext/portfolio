## ✨ Features

- 🎆 **3D Galaxy Particle System** — Interactive Three.js background with 15,000+ particles that respond to mouse movement
- 🎨 **Glassmorphism Design** — Modern frosted-glass UI with backdrop-blur effects and gradient borders
- ⌨️ **Typing Animation** — Dynamic text typing effect in the hero section
- 📊 **Animated Skill Bars** — Smooth progress bars with percentage indicators for each technology
- 🔢 **Counter Animation** — Stats counters that animate when scrolled into view
- 🎭 **AOS Scroll Animations** — Elements animate on scroll with staggered delays
- 📱 **Fully Responsive** — Seamless experience from mobile to 4K displays
- 🌙 **Dark Mode Aesthetic** — Premium dark theme with carefully curated color palette
- ⚡ **Performance Optimized** — Lightweight vanilla HTML/CSS/JS, no heavy frameworks

## 🏁 Getting Started

### Prerequisites

No build tools required! This is a pure **static site** — just open it in your browser.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/callmezext/portfolio.git

# Navigate to the project
cd portfolio

# Open in browser (or use any local server)
# Option 1: Simply open index.html in your browser

# Option 2: Use VS Code Live Server extension

# Option 3: Use Python's built-in server
python -m http.server 3000

# Option 4: Use Node.js serve
npx serve .
```

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```
